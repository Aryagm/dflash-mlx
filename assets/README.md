# Assets

This directory contains generated marketing and documentation assets. Runtime contributors can ignore it unless they are updating README visuals or demo videos.

- `benchmark-chart.png` and `qwen-comparison-chart.png` are generated by `scripts/generate_benchmark_chart.py` and `scripts/generate_qwen_comparison_chart.py`.
- `demo-comparison*.mp4` are generated demo videos for social/README use.
- `captures/*.json` are source captures used by the demo-video scripts.

Regenerate charts with:

```bash
uv run --extra charts python scripts/generate_benchmark_chart.py
uv run --extra charts python scripts/generate_qwen_comparison_chart.py
```
